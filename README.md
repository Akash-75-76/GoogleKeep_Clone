# Google Keep Clone (MERN)

A Google Keep–inspired notes application built using the MERN stack. Users can create, edit, and delete notes using a clean and responsive UI. The frontend is deployed on AWS S3 while the backend handles REST APIs and MongoDB operations.

Tech Stack: React, Material UI, Node.js, Express.js, MongoDB, AWS S3

Features: Create, update and delete notes, responsive UI, RESTful APIs, MongoDB data storage, frontend hosted on AWS S3

Project Structure: client (React frontend), server (Node.js & Express backend)

Setup: Clone the repo → `git clone https://github.com/your-username/google-keep-clone.git` → install dependencies using `cd server && npm install` and `cd ../client && npm install` → create `server/.env` with `MONGO_URI` and `PORT=5000` → run backend using `npm start` and frontend using `npm start`

Deployment: Frontend built using `npm run build` and deployed on AWS S3 as a static website; backend runs separately using Node.js and Express
